제17회 광안리어방축제 자원봉사 모집

모집기간 : 2017. 3. 6.(월) ~ 3. 25.(토) 18:00 마감 (20일간)

활동기간 : 2017. 4. 28.(금) ~ 4. 30.(일)

모집인원 : 총 198명

모집대상 : 만 16세 고등학생 이상 대학생 및 일반인

활동내용 : ▷4/29.(토) 거리퍼레이드 지원팀(100명)은 별도 모집 예정

  • A팀 ('맨손으로 활어잡기' 등 행사장 운영 지원 : 46명)
  • B팀 ('어방민속마을'내 체험부스 운영 지원 : 41명, 통역 4명 포함
  • C, D팀 (지정된 위치에서 방문객 숫자 카운팅 : 54명)
  • E팀 (축제안내소 근무, 홍보물 배부 등 : 57명, 통역 6명 포함)

접수방법 : 자원봉사포털(www.1365.go.kr)에 자원봉사 신청 후
광안리어방축제 홈페이지(http://festival-eobang.suyeong.go.kr)에서 신청

  • 축제 홈페이지 자원봉사모집 신청페이지에서 직접 신청 입력
  • 축제 후 참여증서는 우편으로 방송되오니 정확한 주소(우편번호 포함)작성 요망
  • 신청서 작성 시 원하는 날짜와 근무팀 기입 필히 요망
    ※ 신청한 날짜와 근무팀은 상황에 따라 조정될 수 있음

선발 일정 및 방법 : 2017. 3. 27.(월) ~ 3. 30.(목) / 서류심사(신청서)

결과발표 : 2017. 3. 31.(금) 18:00, 홈페이지 (http://festival-eobang.suyeong.go.kr)

문의처 : 수영구청 문화공보과(☎610-4062, 4049)

※ 자원봉사 실적확인서를 방급받으실 분은 반드시 자원봉사포털(www.1365.go.kr)에 회원가입 하여야 하며, 기존 가입자는 재가입할 필요 없음
